The **Blockchain-Based Lost and Found System** is a simple application designed to help students securely register lost or found items within a college campus. The main goal of the project is to provide a **transparent, tamper-proof, and trustworthy** method for tracking items such as ID cards, laptops, books, bags, pendrives, etc.

This project uses a **blockchain-like structure**, where every item entry (lost or found) is stored inside a block. Each block contains the item details, timestamp, and the hash of the previous block, making the entire chain **immutable** and secure. Once data is added, it cannot be edited or deleted, ensuring trust and transparency.

The system provides simple menu options:

* Register a **Lost Item**
* Register a **Found Item**
* Search for Item by ID
* View complete blockchain records

Whenever a student reports a lost or found item, a new block is created and added to the blockchain. Searching is easy â€” students can type the item ID to see if someone has found their item. This eliminates confusion and prevents misuse of information.
